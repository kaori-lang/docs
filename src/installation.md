# Installation

## Prerequisites

- [Rust](https://rustup.rs/) 1.75 or later

## Building from Source

Clone the repository and build with Cargo:

```bash
git clone https://github.com/you/kaori
cd kaori
cargo build --release
```

The binary will be at `target/release/kaori`.

## Verifying the Installation

```bash
kaori --version
```
